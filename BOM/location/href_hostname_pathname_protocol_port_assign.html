you can use window.location or location,if you want to see more details goto browser console and write console.log(location) or location

1. Getting current page URL href by window.location.href

2. Getting domain name of the web host by window.location.hostname

3. Getting path and file name of current page by location.pathname

4. Getting web protocol by window.location.protocol

5. Getting port by window.location.port

6. Load a new document by window.location.assign("https://www.youtube.com/watch?v=FRg3vcyNpjk") or window.location = "https://www.youtube.com/watch?v=FRg3vcyNpjk". "?v=FRg3vcyNpjk" is a parameter

7. To search the parameter use window.location.search and you have to know also new URLSearchParams

let parameter = new URLSearchParams(window.location.search).get("v")
